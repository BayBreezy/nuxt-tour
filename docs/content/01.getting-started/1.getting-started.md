Everything you need to add guided tours to your Nuxt app - one module install, one component, one composable.

## Installation

::prose-steps
:::prose-step
### Install the package

::prose-pm-install{name="nuxt-tour"}
::
:::

:::prose-step
### Register the module

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-tour"],
});
```

That's it. The `<VTour />` component and the `useTour()` composable are auto-imported everywhere in your app.
:::
::

## Module options

Pass options under the `tour` key in your `nuxt.config`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-tour"],
  tour: {
    prefix: "V",
    injectCSS: true,
    storagePrefix: "nt",
    storageVersion: undefined,
  },
});
```

### Options reference

:::prose-field-group{variant="all"}

  ::prose-field
  ---
  name: prefix
  type: string
  defaultValue: V
  ---
  Prefix for the auto-registered component name. With the default `"V"` the component is registered as `<VTour />`.
  ::

  ::prose-field
  ---
  name: injectCSS
  type: boolean
  defaultValue: true
  ---
  Inject the default `tour.css` stylesheet. Set to `false` if you want to provide your own styles.
  :: 

  ::prose-field
  ---
  name: storagePrefix
  type: string
  defaultValue: nt
  ---
  Prefix used for localStorage keys. A tour named `"onboarding"` is stored under the key `"nt-onboarding"`.
  ::

  ::prose-field
  ---
  name: storageVersion
  type: string
  ---
  Global storage version string. Bump this value to force all previously-completed tours to show again — useful when you make significant changes to a tour's content.
  ::

:::
