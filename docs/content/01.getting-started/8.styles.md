## Default stylesheet

nuxt-tour injects a single `tour.css` file with sensible defaults. It uses **CSS custom properties** (`--nt-*`) so you can override any value without touching the source file or ejecting anything.

To disable the default stylesheet and supply your own:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  tour: { injectCSS: false },
});
```

## Customising with CSS variables

All design tokens are declared on `#nt-tooltip`. Override whichever you need in your global CSS:

```css [assets/css/tour-theme.css]
#nt-tooltip {
  --nt-bg: #1e1e2e;
  --nt-text: #cdd6f4;
  --nt-subtext: #a6adc8;
  --nt-border-color: #45475a;
  --nt-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);
  --nt-radius: 10px;

  --nt-btn-bg: #cba6f7;
  --nt-btn-color: #1e1e2e;
  --nt-btn-radius: 8px;

  --nt-highlight-color: #cba6f7;
}
```

::prose-callout{variant="tip" title="No build step required"}
No Sass, no ejecting — just CSS. The variables are scoped to `#nt-tooltip` so they won't leak into the rest of your app.
::

## Full default stylesheet

::prose-collapsible{openTitle="Hide stylesheet" closeTitle="Show full stylesheet"}
```css [tour.css]
[data-hidden] {
  display: none !important;
}

/* Transitions */
.fade-nt-tooltip-enter-active,
.fade-nt-tooltip-leave-active {
  transition: opacity 0.2s ease, transform 0.2s ease;
}
.fade-nt-tooltip-enter-from,
.fade-nt-tooltip-leave-to {
  opacity: 0;
  transform: scale(0.97);
}

/* Backdrop */

  position: fixed;
  inset: 0;
  background-color: var(--nt-backdrop-bg, rgba(0, 0, 0, 0.5));
  z-index: var(--nt-backdrop-z, 9998);
}

/* Tooltip */
#nt-tooltip {
  position: relative;
  background-color: var(--nt-bg);
  border: 1px solid var(--nt-border-color);
  padding: var(--nt-padding);
  box-shadow: var(--nt-shadow);
  border-radius: var(--nt-radius);
  z-index: var(--nt-z-index);
  max-width: var(--nt-max-width);
  display: flex;
  flex-direction: column;
  gap: var(--nt-gap);
}

/* Centred tooltip (steps with no target) */
.nt-center {
  position: fixed !important;
  top: 50% !important;
  left: 50% !important;
  transform: translate(-50%, -50%) !important;

/* Actions */
.nt-actions {
  display: flex;
  align-items: center;
  gap: 8px;
  margin-top: 10px;
}
.nt-actions button {
  display: inline-flex;
  align-items: center;

  font-weight: var(--nt-btn-font-weight);
  transition: var(--nt-btn-transition);

  cursor: pointer;
  border: 1px solid transparent;

}

  opacity: 0.85;
}

  background-color: transparent;

  border-color: transparent;

  background-color: var(--nt-border-color);

  background-color: var(--nt-btn-bg);

}

/* Highlight */
.nt-highlight {
  outline: 1px solid var(--nt-highlight-color, #0ea5e9);
  outline-offset: var(--nt-highlight-offset, 8px);
  border-radius: var(--nt-highlight-radius, 4px);
}
```
